# Hypothesis Testing

```osmosis
What is a null hypothesis?
***
A statement that there is no effect or no difference; the default assumption to be tested.
```

```osmosis
What is a p-value?
***
The probability of observing results at least as extreme as the data, assuming the null hypothesis is true.
```

```osmosis
What does it mean to reject the null hypothesis?
***
The evidence suggests the observed effect is unlikely due to chance alone (typically p < 0.05).
```
