# Installation

## From Community Plugins

!!! note "Coming soon"
    Osmosis will be available in the Obsidian Community Plugins directory once it reaches stable release.

1. Open **Settings > Community Plugins > Browse**
2. Search for "Osmosis"
3. Click **Install**, then **Enable**

## Manual Installation

1. Download the latest release (`main.js`, `manifest.json`, `styles.css`) from [GitHub Releases](https://github.com/osmosis-md/osmosis/releases)
2. Create the folder `.obsidian/plugins/osmosis/` in your vault
3. Copy the downloaded files into that folder
4. Open **Settings > Community Plugins** and enable "Osmosis"

!!! tip
    After enabling, you'll see two new icons in the left sidebar: a brain icon (mind map) and a graduation cap (study dashboard).
